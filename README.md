## 🛠️ Top DevOps Interview Topics Covered

This repo covers every topic asked in modern DevOps interviews:

**Operating Systems & Linux**
Linux internals, processes, signals, file systems, permissions, systemd, cgroups, namespaces, networking commands, troubleshooting

**Networking**
OSI model, TCP/IP, DNS, HTTP/HTTPS, TLS/SSL, load balancers, reverse proxy, CDN, VPN, firewalls, iptables

**Containers & Docker**
Container internals, Dockerfile best practices, multi-stage builds, image scanning, Docker Compose, networking, volumes, registry

**Kubernetes**
Architecture, Pods, Deployments, Services, Ingress, ConfigMaps, Secrets, RBAC, Network Policies, HPA, Cluster Autoscaler, Helm, Kustomize, Operators, EKS, AKS, GKE

**Infrastructure as Code (IaC)**
Terraform modules, state management, workspaces, Ansible playbooks, CloudFormation, Pulumi

**CI/CD & GitOps**
Jenkins pipelines, GitHub Actions, GitLab CI, AWS CodePipeline, ArgoCD, FluxCD, Blue-Green, Canary, Rollback strategies

**Cloud Platforms**
AWS (EC2, EKS, VPC, IAM, RDS, S3, Lambda, CloudFormation), Azure (AKS, ARM, App Service), GCP (GKE, Cloud Run, BigQuery)

**Monitoring & Observability**
Prometheus, Grafana, Loki, ELK Stack, AlertManager, OpenTelemetry, Datadog, New Relic, distributed tracing

**Security (DevSecOps)**
SAST, DAST, SCA, Trivy, SonarQube, OWASP Top 10, secrets management (Vault, AWS Secrets Manager), IAM, RBAC

**Programming & Scripting**
Bash, Python for automation, YAML, JSON, Groovy (Jenkinsfile), HCL (Terraform)
